# HTM5

Hanh Trinh Magellan 5 Application

**Type**: Web-based application

**Backend**: Colyseus + MongoDB

**Frontend**: React + Redux + Material UI

### Usage

#### Run backend locally

```bash
cd server
yarn
yarn start
```

#### Run frontend locally

```bash
cd client
yarn
yarn dev
```

#### Build frontend

```bash
yarn build
```
