# htm5

Hanh Trinh Magellan 5 Application

### Usage

#### Run backend locally

```bash
cd server
yarn
yarn start
```

#### Run frontend locally

```bash
cd client
yarn
yarn dev
```

#### Build frontend

```bash
yarn build
```
